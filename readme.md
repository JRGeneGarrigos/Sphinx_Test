Comentarios:
^^^^^^^^^^^^
+ He empleado 'Conda' para la creación del entorno virtual. Con la finalidad de que haya trazabilidad con las librerías y versiones respectivas, que he empleado en el proyecto. Con el documento 'requeriments.yaml' se puede autogenerar (generar de forma automática) el entorno utilizado.
+ Uso de de 'Git' como repositorio y 'GitHub' para poder acceder desde la nube al mismo.
+ Uso de 'Sphinx' como herramienta de generación automática de HTML, CSS y JS. Empleando 'Notepad ++' para la edición de los archivos de configuración.
+ Empleo de 'Git Bash' como herramienta para la gestión-administración general del proyecto.
